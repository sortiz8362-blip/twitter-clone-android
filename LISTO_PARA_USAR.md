### OPCIÓN 1: Ejecutar EN EMULADOR (5 min)

```bash
1. Abre Android Studio
2. File → Open → C:\ProyectosGitHub\clonx
3. Espera a que Gradle sincronice
4. Presiona el botón verde ▶ Run
5. ¡LA APP ABRE EN EL EMULADOR!
```

### OPCIÓN 2: Generar APK (10 min)

```bash
1. Ejecuta como en OPCIÓN 1 (asegúrate que funciona)
2. Build → Build Bundle(s) / APK(s) → Build APK(s)
3. Espera 5-10 min
4. El APK estará en: app/build/outputs/apk/debug/app-debug.apk
5. Comparte el APK
```

### OPCIÓN 3: Instalar en tu teléfono (15 min)

```bash
1. Ejecuta OPCIÓN 2 (genera el APK)
2. adb install app/build/outputs/apk/debug/app-debug.apk
3. La app aparece en tu teléfono
4. ¡Abre y úsala!
```
