## 🚀 EMPEZAR AHORA

### 1️⃣ Instala Java

**Descarga:**
```
https://www.oracle.com/java/technologies/javase-jdk11-downloads.html
```

Descarga el archivo para Windows (64-bit) .exe

**Instala:**
- Haz doble clic
- Presiona Siguiente × 5
- Presiona Instalar
- **REINICIA TU PC**

**Verifica:**
Abre PowerShell y escribe:
```powershell
java -version
```

Si ves la versión → ✅ Listo

---

### 2️⃣ Compila el APK

Abre PowerShell (Windows+X) y escribe:

```powershell
cd C:\ProyectosGitHub\clonx
.\gradlew.bat assembleDebug
```

Verás:
```
Downloading gradle...
Building...
BUILD SUCCESSFUL ✅
```

**Espera 10 minutos** (primera vez descarga dependencias)

---

### 3️⃣ Instala en teléfono

**Conecta teléfono por USB**

En teléfono: Ajustes → Opciones de desarrollador → Depuración USB ✓

**En PowerShell:**
```powershell
adb install app\build\outputs\apk\debug\app-debug.apk
```

Verás:
```
Success ✅
```
