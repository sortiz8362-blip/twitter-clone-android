## ⚠️ SI ALGO FALLA

### Error: "Gradle sync failed"
```
Solución:
1. Build → Clean Build Folder
2. File → Sync Now
3. Build → Rebuild Project
```

```
Solución:
1. File → Sync Now
2. Build → Rebuild Project
```
